# Recipe World

A modern recipe application built with React and TypeScript.

## Preview

![Preview](./public/preview.png)

## Features

- Browse recipes
- Filter recipes by category
- Client-side pagination
- Featured recipes section
- Individual recipe pages
- Responsive layout
- Data fetching from a mock REST API (JSON Server)

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- SCSS
- JSON Server
- axios

## Project Structure

- Reusable UI components
- API layer
- Typed models

## Installation

```bash
git clone https://github.com/zkuriansy/recipe-world.git
cd recipe-world
npm install
```

## Running the project

Start the client

```bash
npm run dev
```

Start the mock API

```bash
npm run server
```

Open:

```
http://localhost:5173
```
